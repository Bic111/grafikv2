# Makieta UI: Ustawienia

- **Zakładka 1: Święta**
  - [Tabela] Lista świąt: Data, Nazwa, Akcje (Edytuj, Usuń)
  - [Formularz] Dodaj nowe święto: pole daty, pole nazwy, checkbox "Sklep zamknięty"
- **Zakładka 2: Wymagania Obsadowe**
  - [Selektor] Wybierz typ dnia (Powszedni, Weekend, Święto)
  - [Tabela] Lista zmian: Nazwa zmiany, Rola, Min. obsada, Docelowa obsada, Max. obsada
- **Zakładka 3: Parametry Generatora**
  - [Selektor] Wybierz profil (Domyślny, Sezonowy, Nocny)
  - [Formularz] Wagi optymalizacji: suwak "Preferencje pracownika", suwak "Rotacja zmian"
